# FocusFlow Web

React + Vite ile geliştirilmiş çok bölümlü çalışma planlayıcısıdır. Veriler LocalStorage'da saklanır.

## Özellikler

- Görev ekleme, listeleme, güncelleme, silme ve tamamlama
- Görev arama; tümü, bekleyen ve tamamlanan filtreleri
- Önceki/sonraki ay kontrollerine ve gün seçimine sahip görev takvimi
- Ders kartları, ders bazlı ilerleme oranları ve ders görevleri
- Yeni ders ekleme
- Responsive masaüstü ve mobil arayüz

## Çalıştırma
```bash
npm install
npm run dev
```

Public GitHub repository ve Netlify bağlantısı, hesap sahibi tarafından yayınlandıktan sonra teslim formuna eklenmelidir.

## Netlify

- GitHub bağlantısında build command: `npm run build`
- Publish directory: `dist`
- Manuel yüklemede kaynak klasörü yerine `dist` klasörünün içeriğini yükleyin.
