# FocusFlow Web

React + Vite ile geliştirilmiş görev yönetimi uygulaması. Görev ekleme, listeleme, güncelleme, silme, tamamlama ve arama özellikleri içerir; veriler LocalStorage'da saklanır.

## Çalıştırma
```bash
npm install
npm run dev
```

Public GitHub repository ve Netlify bağlantısı, hesap sahibi tarafından yayınlandıktan sonra teslim formuna eklenmelidir.
